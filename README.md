# Trabalho-pratico

Gerador de Relatórios de Notas



// narrativa sobre o trabalho 

O trabalho “Gerador de Relatórios de Notas” consiste em desenvolver um programa que leia as notas dos alunos a partir de um ficheiro, organize essa informação e produza um relatório com os principais resultados. Depois de carregar os dados, o programa guarda as notas num vetor e faz vários cálculos, como a média, a nota mais alta, a mais baixa e o número total de alunos. Com base nessas contas, gera um relatório final que resume o desempenho geral. No fundo, o objetivo é automatizar o processo de análise das notas e apresentar a informação de forma clara e rápida.
